# PyExplain

PyExplain is a Streamlit-based Python learning assistant that helps beginners understand code instead of just copying it.

Features:
- Concept Detection
- Line-by-Line Code Explanation
- Syntax Checking
- Difficulty Analysis
- Code Summary
- Learning Tips
- Learning Roadmap
- Quiz Generator
- Interview Question Generator
- Downloadable Analysis Reports

Built using:
- Python
- Streamlit
